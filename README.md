## üì¶ Installation

- Install SukiUI Nuget Package

  Avalonia 0.11 -> SukiUI 4.0.0.0 <br/>
  Avalonia 0.10 -> SukiUI 3.0.0.0

- Reference SukiUI in your App.axaml file

```
<Application ...>
     <Application.Styles>
        ...
        <StyleInclude Source="avares://SukiUI/Theme/Index.xaml"/>
        ...
    </Application.Styles>
</Application>
```
